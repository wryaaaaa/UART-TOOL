# 截图说明

请手动截取以下场景的截图并放入此文件夹：

## 建议截图

1. **主界面-接收数据** → `01-receive-data.png`
   - 连接串口后，接收区显示实时数据的画面（含时间戳、ASCII 模式）

2. **趋势图-多变量追踪** → `02-trend-chart.png`
   - 配置了多个变量（如 Err, Out, SET），趋势图显示多条折线的画面

3. **HEX 模式** → `03-hex-view.png`
   - 切换到 HEX 显示模式的画面

4. **三维空间-立方体** → `04-3d-cube.png`
   - 3D 模式下显示立方体模型的画面

5. **三维空间-轨迹线** → `05-3d-trail.png`
   - 3D 模式下显示轨迹线和坐标点的画面

6. **导出功能** → `06-export.png`
   - 点击导出 TXT/JSON 后的下载画面

7. **分屏模式** → `07-split-view.png`
   - 拖拽标签页创建分屏的画面

截图后提交到 GitHub：
```bash
git add screenshots/
git commit -m "docs: 添加应用截图"
git push
```
